## üí° Usage

These emoji files can be imported into Slack or other platforms that support custom emojis.

### File Format

- PNG images: Most emojis
- GIF animations: Animated emojis

### Importing to Slack

#### Manual Upload (One by One)

1. Go to your Slack workspace settings
2. Navigate to **Customize** ‚Üí **Emoji**
3. Click **Add Custom Emoji**
4. Upload the desired image file
5. Set the emoji name (shortcode)

#### Bulk Upload (Recommended)

For uploading multiple emojis at once, use the **Neutral Face Emoji Tools** browser extension:

**[Neutral Face Emoji Tools](https://chromewebstore.google.com/detail/neutral-face-emoji-tools/anchoacphlfbdomdlomnbbfhcmcdmjej)** - Chrome/Firefox Extension

- Upload up to 100 emojis in one go
- Simple drag-and-drop interface
- Automatically uses filename as emoji name
- Available for Chrome and Firefox

**How to use:**
1. Install the extension from [Chrome Web Store](https://chromewebstore.google.com/detail/neutral-face-emoji-tools/anchoacphlfbdomdlomnbbfhcmcdmjej) or [Firefox Add-ons](https://github.com/Fauntleroy/neutral-face-emoji-tools)
2. Navigate to your Slack's `/customize/emoji` page
3. Find the new "Bulk Emoji Uploader" section
4. Drag and drop multiple image files (ensure filenames match desired emoji names)
5. Upload automatically
